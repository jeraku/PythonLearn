Running the redis from docker file. 

1. Build the Docker Image
>> docker build -t my-redis-container .

2. Run the Container
>> docker run -d --name redis-instance -p 6379:6379 my-redis-container

3. Verify Redis Installation
>> docker exec -it redis-instance bash

4. Persist Redis Data (Optional)
>> docker run -d --name redis-instance -p 6379:6379 -v ~/redis_data:/data my-redis-container redis-server --appendonly yes
